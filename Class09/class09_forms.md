# Forms  *(pgs 144-175)*

HTML forms give you a set of elements to collect data from your users

## Form Controls

**Adding Text**

- text input
- password input
- Text area (multi-line)

**Making Decisions**

- radio buttons
- checkboxes
- dropdown menus

**Submitting Forms**

- Submit button
- Image button
- Upload files

## Form Structure `<form>`

Every form structure requires a `action` and a `method`
